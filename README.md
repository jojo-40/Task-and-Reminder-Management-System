# 📌 Daily Task & Reminder Manager

## 📖 Project Description
This project is a C++ console-based application designed to manage daily tasks and reminders.  
It allows users to add tasks, view task lists, and save/load data using file handling.

---

## 🚀 Features
- Add new tasks with ID, title, and time
- Display all saved tasks
- Save tasks to a file (persistent storage)
- Load tasks automatically from file
- Menu-driven console interface

---

## 🧠 Concepts Used
- Classes and Objects
- Inheritance (Base and Derived classes)
- Polymorphism (Function overriding)
- Friend Function / Friend Class
- File Handling (reading and writing files)
- Menu-driven programming
- Vectors for dynamic storage

---

## 💾 File Handling
- `ofstream` → Used to write/save data into file
- `ifstream` → Used to read/load data from file
- File name used: `tasks.txt`

---

## ▶️ How to Run
1. Open the project in Dev-C++
2. Compile and run `main.cpp`
3. Use the menu to add or view tasks
4. Data will automatically be saved in `tasks.txt`
